# Birthday Cake Blow 🎂

An interactive birthday cake web app with a microphone-based candle-blowing effect.

## Getting Started

```bash
npm install
npm start
```

The app opens at **http://localhost:3000**

## Features

- 🕯️ 29 animated candles with flickering flames
- 🎤 Microphone detection — blow into your mic to extinguish all candles at once
- 🎉 Confetti celebration when all candles are blown out
- 💌 Leave birthday wishes with a modal form

## Build for Production

```bash
npm run build
```

Output is placed in the `dist/` folder — ready to deploy to Vercel, Netlify, or any static host.

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Import the repo on [vercel.com](https://vercel.com)
3. Vercel auto-detects Vite — no configuration needed
